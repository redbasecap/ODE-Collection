GitHub Pages

Direkt deploybar: keine Build-Pipeline, keine Runtime-Abhaengigkeiten.

Lege die Dateien in ein GitHub-Repository, aktiviere Pages fuer den Branch oder nutze den mitgelieferten Workflow. Die App laeuft als reine statische Website.

01 Repository pushen

Root-Dateien inklusive index.html, styles.css und script.js committen.

02 Pages aktivieren

GitHub Settings -> Pages -> Source: GitHub Actions oder Branch root auswaehlen.

03 ODE Library teilen

Die generierte URL zeigt die komplette interaktive Sammlung mit Simulationen.

Educational ODE simulations only. Medizinische und finanzielle Modelle sind keine Diagnose, Therapie-, Investment- oder Entscheidungsberatung.
